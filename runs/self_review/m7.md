# M7 — auto-critique adversariale et corrections

Date d’exécution : 2026-07-18. M7 est testé sur une géométrie synthétique ;
aucun fit réel n’est autorisé.

## Vérification exécutée

- Tests exécutés : `3 passed`.
- Ils établissent `R_i = r'' / A_i`, l’agrégation exacte des conductances,
  l’unicité du paramètre `r''`, le refus d’une géométrie non acceptée et le
  verrou M4 avant toute identification réelle.

## Posture du juge hostile

**Quelle affirmation ne survivrait pas à un examen ?** Des surfaces par
orientation ne rendent pas les pertes par orientation identifiables avec une
seule température extérieure. Correction : le code additionne les
conductances et le message d’identifiabilité est stocké dans chaque candidat.

**Quel chiffre est affiché sans incertitude ?** Les valeurs de test sont des
constantes synthétiques, pas des mesures de bâtiment. Correction : aucune
surface ou résistance réelle n’est publiée sans revue M6 et propagation de son
incertitude.

**Quelle hypothèse est enterrée dans le code ?** Toutes les branches voient le
même `Tout`; des effets solaires/orientation ne sont pas ajoutés. Correction :
c’est explicitement la raison de l’agrégation, pas une simplification cachée.

**Qu’est-ce qui marche uniquement sur ce jeu de données précis ?** Rien dans
la formule synthétique; la future association aux plans Pleiades et Tin A est
spécifique et reste non validée. Correction : les barrières M4/M6 empêchent le
passage au réel.

**Où ai-je écrit « ça marche » sans l’avoir exécuté ?** Nulle part : les trois
tests sont exécutés. Aucun fit géométrique réel n’est revendiqué.

## Décision de sortie M7

M7 est disponible comme structure expérimentale contrainte, mais son entrée
réelle est bloquée jusqu’à validation humaine M6 et reproduction M4 validée.
